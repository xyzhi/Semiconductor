# PN 结教学场景整体需求

## 1. 项目目标

本项目为 Unity 2022 内置渲染管线的 PICO VR 半导体教学项目。

`Scene1` 是“PN 结”教学章节，用于展示晶圆、PN 结、P 型半导体和 N 型半导体。用户通过右侧模型按钮切换教学模型，并使用缩放动画表现模型之间的进入和退出关系。

## 2. 场景流程

### Start 场景

- 使用 XR Origin 内的摄像机，不额外创建摄像机。
- 保留头显追踪、手柄追踪、射线和 UI 点击。
- 屏蔽连续移动、转向、传送、抓取移动和攀爬传送。
- 主按钮显示为“pn结”。
- 使用手柄射线点击“pn结”按钮后进入 `Scene1`。

### Scene1 场景

- 默认显示“晶圆”模型。
- PN 结介绍面板显示在模型上方。
- 场景中央作为模型展示区域。
- 右侧显示纵向模型切换栏：
  1. 晶圆
  2. pn结
  3. P型半导体
  4. N型半导体
- 进入场景时，“晶圆”按钮处于当前模型的选中状态。

## 3. 模型组织

教学模型按以下结构制作：

- `WaferModel`：晶圆模型
- `PNJunctionModel`：完整 PN 结模型
  - `PPart`：PN 结中的 P 型半导体部分
  - `NPart`：PN 结中的 N 型半导体部分

P 型半导体和 N 型半导体不再制作成与 PN 结无关的独立模型，而是直接使用 `PNJunctionModel` 内的 `PPart` 和 `NPart`。

晶圆模型和 PN 结模型统一放置在同一个展示锚点下。`PPart` 与 `NPart` 需要保存各自在完整 PN 结中的组合位置，同时支持移动到展示锚点中心。

模型显示状态：

- 晶圆：只显示 `WaferModel`。
- pn结：显示完整 `PNJunctionModel`，`PPart` 和 `NPart` 均位于组合位置。
- P型半导体：只显示 `PPart`，并将其移动到展示区域中心。
- N型半导体：只显示 `NPart`，并将其移动到展示区域中心。

## 4. 切换动画规则

模型顺序定义为：

`晶圆 -> pn结 -> P型半导体 -> N型半导体`

### 向后续模型切换

当目标模型位于当前模型之后时：

1. 当前模型从正常尺寸逐渐放大。
2. 当前模型放大到指定尺寸后消失。
3. 目标模型从较小尺寸开始显示。
4. 目标模型逐渐放大到正常展示尺寸。
5. 动画完成后允许下一次按钮操作。

示例：从“晶圆”切换到“pn结”。

- 晶圆逐渐放大，达到退出尺寸后消失。
- PN 结从较小尺寸逐渐放大至正常尺寸并完成进场。

### 向前序模型切换

当目标模型位于当前模型之前时：

1. 当前模型从正常尺寸逐渐缩小。
2. 当前模型缩小到指定尺寸后消失。
3. 目标模型从较大尺寸开始显示。
4. 目标模型逐渐缩小到正常展示尺寸。
5. 动画完成后允许下一次按钮操作。

示例：从“pn结”切换回“晶圆”。

- PN 结逐渐缩小，达到退出尺寸后消失。
- 晶圆从较大尺寸逐渐缩小至正常尺寸并完成进场。

### 同一模型

- 点击当前已经显示的模型按钮时，不重复播放动画。
- 当前模型和介绍内容保持不变。

### PN 结内部模型切换

本节规则优先于通用的前进、返回缩放规则。

- 点击“P型半导体”按钮时：
  1. 隐藏或移出 `NPart`。
  2. 将 `PPart` 从 PN 结组合位置平滑移动到展示锚点中心。
  3. 居中显示 P 型半导体。
- 点击“N型半导体”按钮时：
  1. 隐藏或移出 `PPart`。
  2. 将 `NPart` 从 PN 结组合位置平滑移动到展示锚点中心。
  3. 居中显示 N 型半导体。
- 从 P 型或 N 型切换回“pn结”时：
  1. 当前居中的部分移动回其 PN 结组合位置。
  2. 恢复另一部分。
  3. 最终完整显示由 `PPart` 和 `NPart` 组成的 PN 结。
- 在 P 型与 N 型之间直接切换时，当前部分退出中心，目标部分进入中心。
- 部分模型在移动、缩放过程中保持朝向稳定。

## 5. 按钮交互

- 所有按钮支持 PICO 手柄射线悬停和点击。
- 当前模型按钮需要有明显的选中状态。
- 动画播放期间暂时禁止重复点击，防止多个切换动画同时执行。
- 动画结束后恢复按钮交互。
- 点击按钮时同步更新：
  - 当前模型
  - 当前按钮选中状态
  - 对应模型标题
  - 对应模型介绍内容

## 6. 教学内容

### 晶圆

- 介绍晶圆的基本概念、材料和用途。
- 模型用于展示晶圆外形及其在半导体制造中的基础作用。

### PN 结

- 介绍 P 区与 N 区接触后的载流子扩散。
- 展示耗尽层和内建电场的形成。
- 讲解正向偏置降低势垒、电流容易通过。
- 讲解反向偏置增大势垒、电流近似截止。
- 突出 PN 结的单向导电性。

### P 型半导体

- 介绍受主杂质掺杂。
- 展示空穴作为多数载流子。
- 展示电子作为少数载流子。

### N 型半导体

- 介绍施主杂质掺杂。
- 展示电子作为多数载流子。
- 展示空穴作为少数载流子。

## 7. 动画参数

以下参数应集中配置，便于后续调整：

- 正常展示尺寸
- 放大退出尺寸
- 缩小退出尺寸
- 放大进场初始尺寸
- 缩小进场初始尺寸
- 退出动画时长
- 进场动画时长
- 动画缓动曲线

动画过程中模型中心位置保持稳定，不应穿过摄像机、遮挡右侧按钮或超出舒适视野。

## 8. 预设要求

- 模型、按钮和教学面板尽量通过预设维护。
- 静态 UI 的文字、颜色、尺寸和位置在预设或场景中设置。
- 不使用代码动态创建静态 UI。
- 模型按钮使用同一个按钮预设实例，便于统一修改样式。
- 模型切换逻辑只负责状态与动画，不在代码中写死静态 UI 样式。

## 9. 验收标准

- 进入 `Scene1` 后默认只显示晶圆。
- 点击“pn结”后，晶圆放大消失，PN 结放大进场。
- 点击“晶圆”后，PN 结缩小消失，晶圆缩小进场。
- 点击“P型半导体”后，只显示 PN 结中的 P 部分，并在展示区域居中。
- 点击“N型半导体”后，只显示 PN 结中的 N 部分，并在展示区域居中。
- 从 P 型或 N 型切回“pn结”后，两部分恢复原组合位置并完整显示。
- P 型与 N 型直接切换时不会出现两个部分在中心重叠。
- 快速连续点击不会造成多个模型重叠或状态错乱。
- 任意时刻只显示当前模型状态所需的模型或组成部分。
- 当前模型按钮的选中状态与当前显示模型一致。
- 射线可以稳定点击全部四个按钮。
- XR Origin 不能通过摇杆移动、转向或传送。
- 模型不会遮挡上方介绍面板和右侧按钮。

## 10. 待确认参数

- 晶圆模型和 PN 结组合模型的最终资源及真实比例。
- `PPart`、`NPart` 的组合位置和居中展示位置。
- 模型在展示锚点中的位置和朝向。
- 放大、缩小的具体倍率。
- 单次进入和退出动画的具体时长。
- 是否为模型切换增加音效、粒子或讲解语音。
- P 型、N 型和 PN 结是否需要载流子运动动画。

## 11. 当前动画按钮

右侧动画按钮与 `PNJunctionTeachingAnimator.PlayByIndex` 的索引保持一致：

1. `0` 扩散：电子与空穴向 PN 结附近移动，表现浓度差导致的多数载流子扩散。
2. `1` 复合：电子与空穴在结区缩小消失，表现载流子复合。
3. `2` 耗尽层：固定离子与耗尽层脉冲显示，强调内建电场区域。
4. `3` 正向偏置：耗尽层变窄，电子和空穴更容易越过结区，表现导通趋势。
5. `4` 反向偏置：耗尽层变宽，载流子远离结区，表现截止趋势。
6. `5` 完整演示：依次播放扩散、复合、耗尽层、正向偏置、反向偏置，并在结束后复位。
7. `6` 复位：停止当前动画并恢复到初始显示状态。
